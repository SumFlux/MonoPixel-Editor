# 字体文件夹

将您的字体文件（.ttf 或 .otf 格式）放在此文件夹中。

## 支持的字体格式

- TrueType 字体（.ttf）
- OpenType 字体（.otf）

## 使用方法

1. 将字体文件复制到此文件夹
2. 启动 MonoPixel Editor
3. 选择文本工具
4. 在字体选择对话框中，点击"加载字体"按钮
5. 选择此文件夹中的字体文件

## 推荐字体

对于像素画编辑，推荐使用等宽字体，例如：
- Consolas
- Courier New
- Monaco
- Source Code Pro
- Fira Code

## 注意事项

- 字体文件名应使用英文，避免使用中文或特殊字符
- 字体文件大小建议不超过 10MB
- 加载的字体仅在当前会话中有效，下次启动需要重新加载
