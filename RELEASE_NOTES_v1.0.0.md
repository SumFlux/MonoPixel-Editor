# MonoPixel Editor v1.0.0 - Release Notes

发布日期：2026-02-09

## 🎉 新功能

### 完整的绘图工具集（8种工具）
- ✨ **画笔工具** - 支持 1-7px 笔触大小
- ✨ **橡皮工具** - 支持 1-7px 橡皮大小
- ✨ **直线工具** - 支持 Shift 键锁定角度（水平/垂直/45度）
- ✨ **矩形工具** - 支持 Shift 键锁定正方形，3种填充模式（轮廓/填充/轮廓+填充）
- ✨ **圆形工具** - 支持 Shift 键锁定正圆，3种填充模式
- ✨ **油漆桶填充** - 4-连通泛洪算法
- ✨ **选择工具** - 框选、移动、缩放（8个手柄，最近邻插值）
- ✨ **文本工具** - 智能文本渲染，支持自定义字体

### 智能文本渲染
- ✨ 半角字符智能挤压（45%-55% 动态调整）
- ✨ 全角字符正常显示（CJK、平假名、片假名等）
- ✨ 支持系统字体 + 自定义字体（fonts/ 文件夹）
- ✨ 精确像素级字号控制（setPixelSize）
- ✨ 文本预览拖拽和二次编辑（Enter 键）

### 多图层系统
- ✨ 透明混合（透明像素不遮挡下层）
- ✨ 图层可见性切换（隐藏/显示）
- ✨ 图层锁定功能（防止误编辑）
- ✨ 图层重命名
- ✨ 图层排序（上移/下移）
- ✨ 图层复制和删除

### 撤销/重做系统
- ✨ 50步历史记录
- ✨ 命令模式实现
- ✨ 所有操作可撤销（绘图、图层操作）
- ✨ 快捷键支持（Ctrl+Z / Ctrl+Y）

### 多格式导出
- ✨ **C Array 格式**（.h 文件）- 适用于嵌入式开发
- ✨ **Binary 格式**（.bin 文件）- 二进制数据
- ✨ **PNG 格式**（.png 文件）- 标准图像格式
- ✨ 水平/垂直扫描模式
- ✨ MSB/LSB 位序选择
- ✨ 实时预览（反向解析验证）
- ✨ 反色选项

### 项目管理
- ✨ 保存/加载项目（.mpx 格式）
- ✨ JSON + Base64 编码
- ✨ 未保存提示
- ✨ 新建画布功能
- ✨ 关闭时保存提示

### 现代化 UI
- ✨ 深色主题（370+ 行 QSS 样式）
- ✨ 完整快捷键系统（文件、编辑、视图、工具切换）
- ✨ 实时鼠标坐标显示
- ✨ 缩放级别显示
- ✨ 网格线切换（快捷键 G）
- ✨ 流畅交互体验

### 架构设计

- **Base64 编码** - 项目文件存储

- ⚡ 内存占用小（< 200MB）

## 🎯 使用场景

### 嵌入式开发
**OLED 显示器**（如 SSD1306）:
1. 创建 128x64 画布
2. 绘制图标或界面
3. 导出为 C Array（垂直扫描、MSB First）
4. 在 Arduino/STM32 中使用

**LCD 显示器**（如 Nokia 5110）:
1. 创建 84x48 画布
2. 绘制图形
3. 导出为 C Array（水平扫描、MSB First）
4. 在嵌入式项目中使用

### 像素艺术创作
1. 创建任意尺寸画布
2. 使用完整工具集创作
3. 多图层管理复杂作品
4. 导出为 PNG 分享

## ⌨️ 快捷键

### 文件操作
- `Ctrl+N` - 新建画布
- `Ctrl+O` - 打开项目
- `Ctrl+S` - 保存项目
- `Ctrl+E` - 导出

### 编辑操作
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做

### 视图操作
- `G` - 切换网格线
- `Ctrl+0` - 重置缩放
- `Ctrl+F` - 适应窗口
- `Ctrl++` / `Ctrl+-` - 放大/缩小

### 工具切换
- `1` - 画笔工具
- `2` - 橡皮工具
- `3` - 直线工具
- `4` - 矩形工具
- `5` - 圆形工具
- `6` - 油漆桶填充
- `7` - 选择工具
- `8` - 文本工具
